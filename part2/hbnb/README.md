# HBnB Project

This project is the initial setup for the HBnB application.
It follows a modular architecture with Presentation, Business Logic, and Persistence layers.

## Project Structure

- `app/` - core application package
- `app/api/` - API endpoints
- `app/models/` - business logic models
- `app/services/` - facade layer
- `app/persistence/` - in-memory repository
- `run.py` - application entry point
- `config.py` - project configuration
- `requirements.txt` - required Python packages

## Installation

Install dependencies with:

```bash
pip install -r requirements.txt
